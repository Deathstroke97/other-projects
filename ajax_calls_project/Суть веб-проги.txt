  Используя ajax вызовы (использую Jquery) связаться с API серверами New York Times и Wikipedia. Далее полученный JSON файл использовать как данные для сайта. В данном проекте планирую также использовать API OpenWeatherApp для включения функции добавления погоды в зависимости от входных данных.

Использовано: HTML, CSS, JavaScript, Jquery, Bootstrap 3
